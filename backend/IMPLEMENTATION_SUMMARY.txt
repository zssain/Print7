GETTING STARTED
================================================================================

1. Navigate to project:
   cd /sessions/pensive-zen-gauss/mnt/Print7/backend

2. Install dependencies:
   npm install

3. Start server:
   npm start

   Server runs on: http://localhost:5000
   Health check: http://localhost:5000/health

4. (Optional) Configure Firebase:
   - Copy .env.example to .env
   - Add Firebase credentials
   - Restart server (auto-detection)

THAT'S IT! No database setup needed.

================================================================================
FIREBASE INTEGRATION (OPTIONAL)
================================================================================

The backend works perfectly WITHOUT Firebase:
- Default: In-memory mock database
- All features work identically
- No setup required

To enable Firebase:
1. Get Firebase credentials (project settings)
2. Add to .env file:
   FIREBASE_PROJECT_ID=your-project-id
   FIREBASE_PRIVATE_KEY=your-private-key
   FIREBASE_CLIENT_EMAIL=your-client-email
   FIREBASE_DATABASE_URL=your-database-url
3. Restart server
4. Auto-detected and enabled

Fallback is automatic if any credentials missing.
